# Macabets v0.22 — NFL Power Ratings

This release preserves the tennis application and upgrades the NFL workspace from a market-derived foundation to an independent Team Power Rating Engine.

## NFL v0.22

- All 32 teams have editable starter rating profiles
- Weighted offense, defense, quarterback, coaching, strength-of-schedule and special-teams grades
- Independent Macabets fair spread and fair moneyline
- Home-field adjustment
- Model-versus-market point edge
- Component-level rating breakdown
- Projected score and restrained confidence label

The starter ratings are model priors, not a live statistical feed. They are intentionally editable. Automated weekly data, injuries, quarterback changes and matchup-specific adjustments are future layers. The total remains market-anchored in v0.22.
